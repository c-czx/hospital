# 医院管理系统 - 患者端使用说明

## 1. 概述

患者端是医院管理系统的用户端核心模块，负责患者的预约挂号、个人信息管理、病历查询和费用缴纳等功能。患者作为系统的服务对象，通过该模块可以便捷地获取医疗服务。

## 2. 核心功能

### 2.1 预约挂号

- **科室和医生查询**
  - 浏览医院科室信息
  - 查看医生详细信息和专业特长
  - 查看医生排班情况

- **预约管理**
  - 在线预约挂号
  - 选择预约时间
  - 填写症状描述
  - 查看预约状态

### 2.2 个人信息管理

- **基本信息管理**
  - 查看、更新个人基本信息

### 2.3 诊疗记录查询

- **病历查询**
  - 查看历史病历记录
  - 查看诊断结果
  - 查看治疗方案

- **处方查询**
  - 查看历史处方记录
  - 查看药品清单和用法用量

- **检查结果查询**
  - 查看历史检查记录
  - 查看检查结果

- **医嘱查询**
  - 查看历史医嘱记录

### 2.4 费用管理

- **账单查询**
  - 查看待支付账单
  - 查看历史账单
  - 查看费用明细

- **在线支付**
  - 在线支付账单
  - 查看历史账单

## 3. 使用流程

### 3.1 注册登录流程

1. 打开医院管理系统患者端
2. 点击"注册"按钮
3. 填写个人信息（姓名、年龄、性别、手机号、邮箱）
4. 设置密码
5. 点击"注册"按钮完成注册
6. 使用注册的账号和密码登录系统

### 3.2 预约挂号流程

1. 登录患者端
2. 点击首页的"预约挂号"按钮
3. 选择科室
4. 选择医生
5. 查看医生排班，选择合适的预约时间
6. 填写症状描述
7. 点击"确认预约"按钮
8. 系统生成预约记录，显示预约成功

### 3.3 个人信息管理流程

1. 登录患者端
2. 点击右上角的"个人中心"图标
3. 点击"基本信息"标签，查看和修改个人基本信息
4. 点击"紧急联系人"标签，管理紧急联系人信息
5. 点击"过敏记录"标签，管理过敏药物记录

### 3.4 诊疗记录查询流程

1. 登录患者端
2. 点击左侧菜单"我的病历"
3. 在病历列表页面，查看历史病历记录
4. 点击病历记录中的"查看详情"按钮，查看病历详细信息
5. 在病历详情页面，可以查看诊断结果、处方、检查结果和医嘱

### 3.5 费用管理流程

1. 登录患者端
2. 点击左侧菜单"我的账单"
3. 在账单列表页面，查看待支付账单和历史账单
4. 点击待支付账单中的"支付"按钮，进入支付页面
5. 选择支付方式，完成支付
6. 支付成功后，系统更新账单状态

## 5. 注意事项

1. **账号安全**：保护账号和密码，避免泄露
2. **预约管理**：按时就诊，如需取消预约请提前操作
3. **个人信息**：确保个人信息的准确性，及时更新联系方式
4. **过敏记录**：如实填写过敏药物记录，避免医疗事故
5. **费用管理**：及时支付账单，避免影响后续就诊
6. **隐私保护**：保护个人医疗信息，不得随意分享
7. **系统操作**：熟练掌握系统操作，提高使用效率
8. **反馈建议**：及时反馈系统使用中遇到的问题和建议
